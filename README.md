# inspect_prompting_comparison

This project uses Inspect to compare the accuracy of Sonnet 3.5 (non-reasoning) on GSM8K for when the prompt is stated once, entirely stated twice, and when the system prompt gets repeated followed by the question and the question being repeated.

The results were that Sonnet 3.5 had a 69% accuracy with one prompt, 76% accuracy with the entire prompt repeated, and 77% accuracy with the two parts of the prompt repeated individually.
